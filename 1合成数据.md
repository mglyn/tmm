# subtask1: 合成数据

目标是在本地构建 TMM 实验所需的 synthetic chart 数据集，包括图表图片、底层 manifest、五阶段训练样本、CRS 校验结果和 router holdout 集。

这一步不算力密集，不需要 GPU，也不需要上 AutoDL。默认用本地 CPU 和 `matplotlib` 生成图表即可；AutoDL 留给后面的 LoRA 微调和评测。

## 1.1 本地前置条件

在本地安装轻量依赖即可。如果你已经能跑 Python，可以先执行：

```powershell
python -m pip install pyyaml matplotlib numpy pillow
```

如果你想直接装完整项目依赖，也可以执行：

```powershell
python -m pip install -r requirements.txt
python -m pip install -e .
```

检查项目根目录：

```powershell
Get-Location
Get-ChildItem
```

应能看到：

- `configs/`
- `scripts/`
- `src/`
- `data/`
- `outputs/`
- `tmm_paper/`

## 1.2 小规模验证

先不要直接生成全量数据，先跑 20 张图的验证：

```powershell
python scripts/run_smoke_test.py --config configs/base_experiment.yaml --limit 20
```

确认输出中有：

- `chart_count`
- `stage_counts`
- `router_holdout_count`
- `difficulty_counts`

## 1.3 全量生成

```powershell
python scripts/build_dataset.py --config configs/base_experiment.yaml
```

默认配置来自 `configs/base_experiment.yaml`：

- 合成图表数：`1973`
- 图表类型：`bar / line / pie / scatter / area`
- 场景类别：`10` 类
- 目标训练样本数：`13811`
- 阶段样本数：`S1/S2/S3/S5` 各 `1973`，`S4` 为 `5919`
- router holdout：约 `1200`

manifest：

## 1.5 质量检查

查看汇总：

```powershell
python -c "import json; from pathlib import Path; p=Path('data/synthetic/manifests/summary.json'); print(json.dumps(json.loads(p.read_text(encoding='utf-8')), indent=2, ensure_ascii=False))"
```

重点检查：

- `chart_count` 是否接近 `1973`
- 五阶段 `stage_counts` 是否为 `1973 / 1973 / 1973 / 5919 / 1973`
- `stage3_reasoning_vqa` 是否存在
- `router_holdout_count` 是否非零

## 1.6 CRS 校验说明

`stage3_reasoning_vqa` 会生成 Python reasoning trace，并在构建样本时执行校验：

- 代码能编译
- 代码能执行
- 代码中的 `answer` 与期望答案一致

不通过的 trace 不应进入最终训练集。

## 1.7 同步到 AutoDL

本地生成全量数据后，再把以下内容同步到 AutoDL：

```text
data/synthetic/charts/
data/synthetic/manifests/
```

如果用 git 管理，不建议把大量图片和中间产物直接提交进 git；更稳妥的是压缩后上传：

```powershell
Compress-Archive -Path data/synthetic -DestinationPath synthetic_data.zip -Force
```

上传到 AutoDL 后解压到项目根目录：

```bash
unzip synthetic_data.zip -d data/
```

确认 AutoDL 上存在：

```bash
ls data/synthetic/charts | head
ls data/synthetic/manifests
```
